# Publicar AplicaSinSpam

## Ruta recomendada

1. Crear un repositorio público en GitHub llamado `aplica-sin-spam`.
2. Subir todos los archivos de esta carpeta.
3. En GitHub, entrar a `Settings > Pages`.
4. Elegir `Deploy from a branch`.
5. Seleccionar rama `main` y carpeta `/root`.
6. Guardar.
7. Esperar la URL de GitHub Pages.

## Antes de AdSense

- Cambiar `TU-USUARIO` en `sitemap.xml` por el usuario real de GitHub.
- Publicar y probar que `index.html`, `acerca.html` y `privacidad.html` abran bien.
- Registrar el sitio en Google Search Console.
- Solicitar AdSense cuando ya esté publicado.

## Anuncios

Los espacios laterales ya están preparados visualmente. Cuando AdSense apruebe el sitio, reemplaza los placeholders `ad-rail` por el código real de los bloques de anuncio o usa Auto Ads en el `<head>`.
